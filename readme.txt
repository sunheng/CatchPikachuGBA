Pikachus, I choose you!

You play as a pokeball. Your goal is to catch 5 pikachus to win. Avoid the evil meowths. You win after catching 5 pikachus.

Start the game by pressing enter.

During the game, you can press backspace to restart the game.

Use the arrow keys to move your pokeball.

Gotta catch em' all...but the meowths.	

This can be run using the VBA, a GameBoy Advance emulator using 'make vba'.
